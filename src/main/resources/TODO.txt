# 儿童端 (ChildController) APP移动端
# 儿童端接口 (ChildController) - 移动端
## 基础功能模块
    1. 登录 (POST /api/child/login) - 儿童ID+4位验证码登录
    2. 获取儿童个人信息 (GET /api/child/info) - 获取基本信息
    3. 发送聊天消息 (POST /api/child/chat/send) - 文字/语音输入
    4. 获取聊天记录 (GET /api/child/chat/history) - 获取历史聊天
    5. 紧急呼叫 (POST /api/child/emergency-call) - 触发紧急联系

# 社工端接口 (Web端)
## 基础功能模块 (SocialWorkerController)
    1. 登录 (POST /api/social-worker/login) - 已实现
    2. 登出 (POST /api/social-worker/logout)
    3. 获取个人信息 (GET /api/social-worker/info)
    4. 更新个人信息 (PUT /api/social-worker/info)

## 儿童管理模块 (SocialWorkerController)
    1. 获取儿童列表 (GET /api/social-worker/children/list) - 分页查询，显示ID、姓名、年龄，标注新聊天记录
    2. 获取儿童详情 (GET /api/social-worker/children/{id}) - 显示AI提取的结构化信息
    3. 创建儿童账号 (POST /api/social-worker/children) - 社工预先创建儿童账号
    4. 更新儿童信息 (PUT /api/social-worker/children/{id})
    5. 重置儿童验证码 (POST /api/social-worker/children/{childNo}/reset-code)

## AI聊天模块 (SocialWorkerController)
    1. 获取会话列表 (GET /api/social-worker/chat/sessions) - 获取儿童与AI的会话列表
    2. 发送消息 (POST /api/social-worker/chat/message) - 社工向儿童发送消息
    3. 获取聊天记录 (GET /api/social-worker/chat/records) - 获取指定会话的聊天记录

## 紧急呼叫模块 (SocialWorkerController)
    1. 获取紧急呼叫记录 (GET /api/social-worker/emergency/calls) - 获取儿童的紧急呼叫记录

## 帮扶方案模块 (SocialWorkerController)
    1. 生成AI帮扶方案 (POST /api/social-worker/scheme/generate) - 根据儿童信息生成AI帮扶方案
    2. 获取方案列表 (GET /api/social-worker/scheme/list) - 按儿童ID或状态查询
    3. 获取方案详情 (GET /api/social-worker/scheme/detail/{id})
    4. 修改方案状态 (POST /api/social-worker/scheme/change-status) - 修改帮扶方案的状态

## 帮扶跟踪模块 (SocialWorkerController)
    1. 获取帮扶跟踪日志列表 (GET /api/social-worker/track/logs) - 获取指定方案的帮扶跟踪日志
    2. 添加帮扶跟踪日志 (POST /api/social-worker/track/log) - 添加新的帮扶跟踪日志
    3. 更新帮扶跟踪日志 (PUT /api/social-worker/track/log/{id}) - 更新帮扶跟踪日志内容

## AI分析模块 (SocialWorkerController)
    1. 获取AI分析结果列表 (GET /api/social-worker/ai/analysis/list) - 获取指定儿童的AI分析结果

## AI模拟训练模块 (AiSimulationController)
    1. 获取模拟训练场景列表 (GET /api/social-worker/simulation/scenarios) - 获取可用的训练场景列表
    2. 开始新的训练会话 (POST /api/social-worker/simulation/start) - 开始一个新的训练会话
    3. 发送训练消息 (POST /api/social-worker/simulation/send) - 社工向模拟儿童发送消息
    4. 获取训练会话记录 (GET /api/social-worker/simulation/history/{sessionId}) - 获取指定训练会话的聊天记录
    5. 获取AI指导意见 (GET /api/social-worker/simulation/advice/{sessionId}) - 获取AI对当前训练的指导意见
    6. 结束训练会话 (POST /api/social-worker/simulation/end/{sessionId}) - 结束训练会话并获取总结评估

## 1. 项目概述

CareSync AI 是一个专注于留守儿童情感陪伴与帮扶的项目，构建 "AI 智能陪伴 + 社工精准介入" 的轻量化服务体系。通过安全合规的 AI 交互填补日常情感空白，依托结构化数据支撑社工开展针对性帮扶。

**核心价值**：
- 儿童端：提供有温度、可信赖的即时情感回应和安全的倾诉出口
- 社工端：通过 AI 提炼的结构化数据快速定位儿童需求，实现数据驱动的精准帮扶
- 管理端：形成交互-分析-帮扶-跟踪的闭环数据沉淀，提供可量化的效果评估依据

## 2. 核心技术栈

| 技术层面   | 选型方案              | 选型依据                                                     |
|----------|-------------------|----------------------------------------------------------|
| 后端框架    | SpringBoot 3.x    | 轻量化开发，支持快速集成第三方服务，适配小团队敏捷迭代需求            |
| 数据库     | PostgreSQL 15.x   | 原生支持 JSONB 类型与复杂查询，满足结构化与非结构化数据存储需求          |
| 前端框架    | Vue 3 + Element Plus | 组件化开发效率高，UI 组件适配管理系统场景，学习成本低                 |
| AI 模型    | 定制化儿童专用大模型        | 优化儿童语言理解能力，内置安全规则，避免通用模型的风险隐患              |
| 语音处理    | 阿里云 ASR/TTS      | 识别准确率高，支持儿童语音优化，提供稳定的 API 服务                   |
| 数字人交互   | 对接 livetalking 服务 | 专注数字人实例管理与画面传输，后端仅维护会话关联，降低开发复杂度            |
